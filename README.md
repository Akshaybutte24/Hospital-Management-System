Hospital Management System (Python)

A command-line based Hospital Management System developed using Python. This project helps manage patients, doctors, appointments, and billing records efficiently. It is designed to automate basic hospital operations and reduce manual workload.

Features
• Add, view, update, and delete patient records
• Manage doctor details and availability
• Book and track appointments
• Generate billing information
• Simple menu-driven interface
• File-based data storage

Tech Stack
 Python
 File handling
 OOP concepts
 Flask
